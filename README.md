# 📊 Sales Dashboard (Tableau)

## Overview  
This project is a **Sales Performance Dashboard** built in **Tableau Public** as part of my practice in **Data & Business Analytics**.  
The dashboard provides a comprehensive view of sales metrics, profitability trends, and geographic performance between **2015–2017**.  

The goal of this project is to demonstrate my ability to:  
- Design interactive dashboards  
- Apply business intelligence storytelling  
- Transform raw data into actionable insights  

---

## 🔑 Key Features  
- **KPIs at a glance**:  
  - Revenue: ₹24.91M  
  - Profit: ₹10.46M  
  - Cost: ₹14.46M  
  - Quantity Sold: 84,174  
  - Return Quantity: 1,828  
  - Total Orders: 25,164  

- **Trend Analysis**:  
  - Running totals (YTD, QTD, MTD)  
  - Profit trends with moving averages  
  - Weekly order vs. profit comparison  

- **Geographic Insights**:  
  - Country-wise profit distribution  
    - United States: 32.17%  
    - Australia: 29.42%  
    - United Kingdom: 11.62%  
    - Canada: 7.25%  

- **Visualizations Used**:  
  - Line charts, bar charts, pie charts, bubble charts  
  - Dual-axis comparisons  
  - Interactive filters for order date and categories
